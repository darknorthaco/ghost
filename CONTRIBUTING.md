# Contributing to GHOST

Thank you for helping improve GHOST.

## License and rights

By opening a pull request or otherwise contributing code, documentation, or other materials to this repository, you agree that:

1. **Your contribution** will be licensed under the same terms as the project: the [**PolyForm Noncommercial License 1.0.0**](LICENSE) (and, where applicable, any **commercial license** terms that apply to Dark North Co.’s distribution of your merged work—see [`COMMERCIAL.md`](COMMERCIAL.md)).
2. **You have the right** to submit the contribution (you created it or have permission to license it).
3. **You grant** Dark North Co. and the project maintainers the rights needed to use and redistribute your contribution under those licenses.

If you contribute on behalf of an employer, ensure you are authorized to do so.

## Developer Certificate of Origin (DCO)

By contributing, you certify the [Developer Certificate of Origin](https://developercertificate.org/) version 1.1, using a **Signed-off-by** line in your commit message:

```text
Signed-off-by: Random J Developer <random@developer.org>
```

Use your real name and a reachable email address.

## Process

- Open issues for substantial changes before large PRs when possible.
- Keep PRs focused and match existing code style.
- Do not submit secrets, API keys, or proprietary third-party code you cannot license.

## Doctrine

Please read [`DOCTRINE.md`](DOCTRINE.md) so changes stay aligned with project principles.
