# GHOST Installer Assets

This directory holds optional visual assets for the installer EXE:

- `ghost.ico` — Window icon (256×256 recommended)
- `sidebar_logo.png` — Sidebar brand image (160×80)
- `banner.bmp` — Classic WinXP-style wizard banner (480×60)

These files are optional.  If absent, the installer uses text-only rendering.
Place assets here and rebuild the EXE to include them.
